# ID-Card-generator

A web-based ID card generator built for SMIT (Saylani Mass IT Training) students.
Fill in the form and instantly generate a styled student identity card.

## 🔗 Live Demo
[https://id-card-generator-js-projects.netlify.app/](https://id-card-generator-js-projects.netlify.app/)

## Features
- Full form validation (no empty fields allowed)
- Auto-generated roll number
- Profile photo upload with live preview
- Clean ID card output with all student details
- Fully responsive design

- HTML
